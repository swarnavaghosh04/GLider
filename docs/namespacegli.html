template<std::size_t N>
void gli::initialize ( int  major,
int  minor,
const SDLAttributeArray< N > &  attributes 
)

Initializes all subsystems for SDL2.

This is an overloaded member function, provided for convenience. It differs from the above function only in what argument(s) it accepts.

This, or any other overload of this function, should be called before using any funtions from GLider. It initlializes all the subsystems for SDL2 and sets important attributes.

Remarks
When GLI_DEBUG is defined, this function will set the priority of SDL_LOG_CATEGORY_APPLICATION to SDL_LOG_PRIORITY_DEBUG.
The following 3 attributes will always take up the following values after this function call regardless of the value passed in through attributes:
  • SDL_GL_CONTEXT_PROFILE_MASK = SDL_GL_CONTEXT_PROFILE_CORE
  • SDL_GL_CONTEXT_MAJOR_VERSION = major
  • SDL_GL_CONTEXT_MINOR_VERSION = minor
Parameters
[in]majorthe major version for OpenGL to be initialized. should be greater than 3.
[in]minorthe minor version for OpenGL to be initialized. defaults to 0.
[in]attributesstd::array of std::pair<SDL_GLattr, int> specifying user SDL attribute values.
Template Parameters
Nnumber of std::pair objects inside attributes.
Exceptions
std::invalid_argumentinvalid OpenGL version.
std::runtime_errorSDL2 could not be initialized or GL attributes could not be set.

◆ initialize() [4/4]

void gli::initialize ( int  major = 3,
int  minor = 0 
)

Initializes all subsystems for SDL2 with default SDL2 attributes.

This, or any other overload of this function, should be called before using any funtions from GLider. It initlializes all the subsystems for SDL2 and sets important attributes.

Remarks
When GLI_DEBUG is defined, this function will set the priority of SDL_LOG_CATEGORY_APPLICATION to SDL_LOG_PRIORITY_DEBUG.
This function will initate SDL2 with the following attributes:
  • SDL_GL_CONTEXT_PROFILE_MASK = SDL_GL_CONTEXT_PROFILE_CORE
  • SDL_GL_CONTEXT_MAJOR_VERSION = major
  • SDL_GL_CONTEXT_MINOR_VERSION = minor
  • SDL_GL_DOUBLEBUFFER = 1
  • SDL_GL_DEPTH_SIZE = 32
  • SDL_GL_MULTISAMPLEBUFFERS = 1
  • SDL_GL_MULTISAMPLESAMPLES = 4
  • SDL_GL_ACCELERATED_VISUAL = 1
if you want to have different attributes, use one of the overloads of this function.
Parameters
[in]majorthe major version for OpenGL to be initialized. should be greater than 3.
[in]minorthe minor version for OpenGL to be initialized. defaults to 0.
Exceptions
std::invalid_argumentinvalid OpenGL version.
std::runtime_errorSDL2 could not be initialized or GL attributes could not be set.
